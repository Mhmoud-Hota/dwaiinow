# تطبيق دوائي (Dawai App) - شاشة الترحيب

هذا المشروع هو تطبيق Flutter تم تطويره لتحويل تصميم شاشة ترحيب تطبيق "دوائي" إلى كود برمجي احترافي.

## المميزات التقنية:
- **نظام إدارة الحالة (Cubit):** تم استخدام `flutter_bloc` لإدارة الحالة لضمان سهولة التوسع لاحقاً.
- **التصميم المتجاوب (Responsive Design):** تم استخدام مكتبة `flutter_screenutil` لضمان ظهور الشاشة بشكل مثالي على جميع أحجام الهواتف الذكية.
- **الخطوط (Fonts):** تم استخدام خط `Cairo` من Google Fonts ليتناسب مع اللغة العربية والهوية البصرية للتطبيق.
- **هيكلة المشروع (Clean Architecture):** تم تنظيم الملفات بطريقة تسهل إضافة ميزات جديدة (features) مستقبلاً.

## هيكل المجلدات:
- `lib/core`: يحتوي على الثيمات والأدوات العامة.
- `lib/features/welcome`: يحتوي على منطق وواجهات شاشة الترحيب.
  - `presentation/cubit`: منطق إدارة الحالة.
  - `presentation/welcome_screen.dart`: الواجهة الرئيسية.

## كيفية التشغيل:
1. تأكد من تثبيت Flutter على جهازك.
2. قم بتنفيذ الأمر `flutter pub get` لتحميل المكتبات.
3. قم بتشغيل التطبيق باستخدام `flutter run`.
